# Minimum Wage and Teen Employment: A DiD Example

This is the running example for the book *Git and GitHub for
Social Science Researchers*. It is a small, fully simulated
difference-in-differences study: did a minimum-wage increase
in January 2019 reduce the teen employment rate in treated
counties?

**All data are simulated.** No real licensed or confidential
data are used. The raw data are generated by a seeded script,
so the entire project rebuilds from code alone.

## How to reproduce

```bash
python scripts/00_make_data.py   # build simulated raw data
python scripts/01_clean.py       # clean -> analysis.parquet
python scripts/02_analysis.py    # estimate DiD, write outputs
```

## Layout

```
scripts/   numbered, run in order
data/raw/  simulated raw data        (git-ignored)
data/clean/ derived analysis file    (git-ignored)
output/    tables and figures
```

## Environment

Python 3.10+, with `polars`, `pyfixest`, `pandas`,
`matplotlib`. Pinned versions live in the project lockfile
(see Chapter 8).
